CultivationX
AI-powered developer career growth platform
View on GitHub ↗

A Spring Boot 3.5 modular monolith unifying resume analysis, AI code mentoring, and GitHub/LeetCode activity tracking — turning scattered developer signals into one composite readiness score.

  • Resume intelligence: AI-powered parsing and ATS scoring using Apache Tika/PDFBox and Spring AI, extracting skills and generating personalized learning roadmaps via Groq LLM.
  • AI coding mentor: conversational memory, code review across correctness/performance/security, and optimized-solution generation.
  • Live activity sync: GitHub OAuth and LeetCode GraphQL integration with a scheduled sync engine tracking real developer activity.
  • "LeetGit" pipeline: AI-reviews LeetCode submissions and auto-pushes formatted solutions with complexity analysis and interview notes to GitHub.
